# 河内塔 3D 学习游戏
一个使用 HTML + CSS + JavaScript 实现的单页互动网页游戏，模拟经典河内塔谜题，采用 3D 视觉风格。

---

## 🎮 游戏功能
- **圆盘数量设置**：支持 3-8 个圆盘，实时重置游戏状态
- **点击式移动**：点击选中圆盘，再点击目标柱子即可移动
- **规则验证**：禁止将大盘放在小盘上，无效操作会有警告动画
- **自动解题**：一键演示完整的递归解法，带分步延迟动画
- **步数统计**：实时记录并显示移动步数
- **胜利检测**：所有圆盘移至最右侧柱子后，显示胜利提示

---

## 🛠️ 技术实现
- **HTML**：语义化结构，响应式布局
- **CSS**：
  - 纯 CSS 实现 3D 透视与倾斜效果
  - 圆盘颜色渐变、阴影、圆角样式
  - 警告抖动、自动解题等关键帧动画
- **JavaScript**：
  - 面向对象设计，游戏状态由数组管理（栈模拟）
  - 递归算法实现自动求解
  - 事件监听处理用户交互
  - 动态渲染游戏界面

---

## 📂 项目文件
- `index.html`：游戏主页面结构
- `style.css`：所有样式、3D 效果与动画
- `script.js`：游戏核心逻辑、状态管理与自动求解算法

## 📖 玩法说明
1.  拖动滑块设置圆盘数量（3-8），游戏会自动重置
2.  点击柱子上最上方的圆盘选中，再点击目标柱子放置
3.  只能将圆盘放在空柱子或比它大的圆盘上方
4.  点击「自动解题」可观看完整解法演示
5.  点击「重置」可恢复当前圆盘数的初始状态
